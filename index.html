Python - CheatSheet

1) L'affectation

Une variable est un espace de stockage dans lequel on peut lire et écrire.

Exemple :

                        text = "mot"
print(text)

output :
"mot"

2) Les differents types :

Il y a 15 types principaux en python. Les 5 plus utilisé sont :
-string (str), les chaine de characteres
-integer (int), les nombres entiers
-floatant (float), les nombres à virgules
-boolean (bool), un état (True ou False)
-liste (list), une liste d'element

Exemple :

                        print(type("chaine"))  #string
print(type(1))  # integer
print(type(1.1))  # floatant
print(type(True))  # boolean
print(type([1, 2, "3"]))  # list

output :
<class 'str'>
<class 'int'>
<class 'float'>
<class 'bool'>
<class 'list'>

3) Les operateurs mathematiques :

Ils permettens d'effectuer des operations arithmetiques.

Exemple :

                        a = 1
b = 2
print(a + b)    # addition
print(b - a)    # soustraction
print(a * b)    # multiplication
print(a ** b)   # carre
print(a % b)    # reste de division euclidienne
print("1" + "2")

output :
3
1
2
1
1
"12"

4) Les operateurs de comparaisons :

Ils permettent de comparer 2 valeurs. Ils retournent un boolean.

Exemple :

                        x = 3
y = 5
print(x == y)
print(x != y)
print(x > y)
print(x < y)
print(x >= y)
print(x <= y)
print(x == "3")

output :
False
True
False
True
False
True
False

5) Les operateurs logiques :

Ils permettent de comparer plusieurs elements. Ils retournent un boolean.

Exemple :

                        a = 5
b = 7
print(a == b or a == 7)
print(a != b and a == 7)
print(not(a == b or a == 7))

output :
True
True
False

6) La conditionelle

Elle permet de choisir d'executer certaines actions et pas d'autre.

Exemple :

                        color = "red"
if color == "red":
    print("color is red")
else:
    print("color is not red")

output :
"color is red"

7) La boucle bornée

Elle permet de repeter un bloc d'instruction un nombre fixe de fois.

Exemple :

                        for i in range(3):
    print(i)

output : 
0
1
2

8) La boucle non-bornée

Elle permet de repeter des instructions un nombre indefini de fois (peut être infini).

Exemple :

                        nbr = 0
While nbr != 2:
    print(nbr)
    nbr += 1

output:
0
1

9) Les fonctions

Elle permet de reutiliser un bloc de code dans le programme. Pour cela on nomme le bout de code et on le reutilise en utilisant sont nom.

Exemple :

                        def square(x):
    print(x ** 2)
square(2)

output :
4
